# Coinflip (Hackathon Project)

This repository contains the Coinflip project in:

- `blockathon-bets/`

## Project

**Coinflip — The 50/50 Market**

Friends-only peer sports market (NBA-first) with private groups, head-to-head offers, and verifiable settlement artifacts.

## Start here

```bash
cd blockathon-bets
pnpm install
pnpm dev
```

Then open `http://localhost:3000`.

## Verification

```bash
cd blockathon-bets
pnpm smoke
```

Expected output includes `SMOKE PASS`.
